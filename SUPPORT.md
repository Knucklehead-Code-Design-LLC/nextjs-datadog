# Support

## Questions And Usage Help

The project is currently in design and does not have a supported npm release.
Use GitHub Discussions for usage and design questions.

Use an issue for a reproducible defect or a focused feature proposal. Provide a
minimal reproduction and the relevant Next.js, Node.js, React, Datadog SDK,
OpenTelemetry, hosting platform, and package versions.

## Boundaries

This community project cannot provide Datadog account support, billing help,
service availability guarantees, or support for private Datadog APIs. Contact
Datadog Support for Datadog service and account issues.

Maintainers may close requests that cannot be reproduced, fall outside the
project scope, contain sensitive data, or require unsupported vendor behavior.
Community support is provided on a best-effort basis without a response-time
commitment.

Security vulnerabilities must follow [SECURITY.md](SECURITY.md), not the public
support process.
